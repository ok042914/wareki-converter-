# 西暦・和暦 ドラムロール変換アプリ

## 概要
西暦をドラムロールで選択すると、昭和・平成・令和が連動して表示されるWebアプリ。
インストール不要、スマホブラウザで動作する。

## 機能要件
- 西暦のドラムロールをスクロールすると和暦が連動して動く
- 昭和・平成・令和の3つを常に表示
- 選択中の西暦に対応する元号だけハイライト表示
- 範囲外の元号はグレーアウト表示
- ドラムロールは実際に回転しているアニメーション付き
- 年のみ（月日なし）

## 対応範囲
| 元号 | 西暦 |
|------|------|
| 昭和 | 1926年〜1989年 |
| 平成 | 1989年〜2019年 |
| 令和 | 2019年〜 |

## 動作環境
- スマホブラウザ（iOS Safari / Android Chrome）
- PCブラウザでも動作すること

## 技術スタック（予定）
- HTML / CSS / JavaScript（ライブラリなし）
- 単一HTMLファイルで完結
